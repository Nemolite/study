Требования к курсовому проекту ПМ 07 (МДК 07.01)

1.	Проект создается по изученным веб-технологиям (HTML/CSS/JS/PHP/MySQL)
2.	Проект должен быть Клиент-Серверным приложением
3.	Проект должен соответствовать выбранной тематике
4.	Пользовательская (клиентская) сторона проекта – это отдельный модуль, который включает в себя :
4.1	Адаптивную верстку сайта (возможно применение bootstrap,grid,flex ) 
4.2	Минимум 5 HTML – страниц
4.3	Минимум 1 модальное окно на JavaScript
4.4	Минимум 1 слайдер (без применения библиотек на базе JQuery)
4.5	Минимум 1 форма, с как минимум 5 разными по типу полей ввода (text,number,chebock,radio,select)
5.	Серверная сторона проекта – это отдельный модуль, который включает в себя:
5.1	Серверную логику , написанную на PHP 8 (рекомендуется в ООП стиле )
5.2	 Для взаимодействие с базой данных используется модуль PDO PHP
6.	В качестве хранилища данных – MySQL, в котором должным быть как минимум 3 сущности (таблицы), без учета связывающих (при связи многие ко многим)
7.	Передача данных от клиентской стороны на сервер осуществляется с применением JavaScript (AJAX, асинхронная передача данных, промеси)
8.	Перед передачей данных выполняется валидация данных с использованием регулярных выражений
  

     Запрещено:
1.	Использование кода, сгенерированного нейросетью либо любым другим генератором кода
2.	Запрещается использование библиотеки JQuery
